Connect to Fastmail
Use a read-only token — select Mail — Read access only. This extension never sends, deletes, or modifies any mail.
  1. Go to Fastmail → Settings → API tokens
  2. Click New token and give it a name
  3. Under permissions, enable Mail and set it to Read only
  4. Copy the token (shown once) and paste it below

Your token is encrypted before being stored — it is never saved in plain text.
